Sobre o Projeto

Este projeto de TCC em Desenvolvimento de Sistemas tem como objetivo analisar e visualizar dados multidimensionais da cidade de São Paulo, integrando informações de diversas fontes para criar uma visão holística da dinâmica urbana.

Através de técnicas de análise de dados e visualização geográfica, exploramos cinco eixos principais:

  • Movimentação comercial e econômica
  • Propagação de doenças e saúde pública
  • Concentração de poluentes e qualidade do ar
  • Distribuição de zonas de lazer e cultura
  • Presença de refugiados e imigrantes

O projeto visa auxiliar gestores públicos, pesquisadores e cidadãos a compreenderem as complexas relações entre esses fatores urbanos.

Metodologia

Nosso projeto utiliza uma abordagem técnica integrada para coleta, processamento e visualização de dados.

Coleta de Dados

Integração de APIs públicas (IBGE, CETESB, Secretaria de Saúde), web scraping de portais governamentais e parcerias com ONGs para dados de imigração.

Processamento

Limpeza e normalização dos dados usando Python (Pandas, NumPy), com armazenamento em banco de dados PostgreSQL com extensão PostGIS para dados geográficos.

Análise

Aplicação de algoritmos de machine learning para identificar padrões e correlações entre os diferentes conjuntos de dados.

Visualização

Desenvolvimento de mapas interativos usando Leaflet.js e heatmaps personalizados, com backend em Node.js para servir os dados processados.

Tecnologias

Stack completo: Python (análise), JavaScript (visualização), PostgreSQL (armazenamento), Docker (containerização), Git (controle de versão).

Ética de Dados

Todos os dados pessoais são anonimizados, com especial atenção aos dados sensíveis de saúde e imigração, seguindo a LGPD.
